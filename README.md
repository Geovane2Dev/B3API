# BrInvestidor

## Overview

The BrInvestidor is a simple RESTful API designed to retrieve stock-related data from the Brazilian stock exchange (B3).

## Getting Started

### Prerequisites

Before you begin, ensure you have the following:

- Node.js installed
- npm (Node Package Manager) installed
